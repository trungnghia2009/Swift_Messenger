# Messenger Real Time Chat App

A swift project to create a real time chat application in Swift 5 using Firebase

## Features
- Facebook Login
- Google Sign in
- Email/Pass Registration/Login
- Photo Messages
- Video Messages
- Location Messages
- Real Time Conversations
- Search for Users
- Deleting Conversations
- User Profile
- Dark Mode Support
